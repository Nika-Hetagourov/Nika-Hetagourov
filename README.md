### 🏆 Projets Mis en Avant

#### 🏥 [Prédiction des Maladies Cardiovasculaires](https://github.com/Nika-Hetagourov/Prediction-Maladies-Cardiovasculaires-R)
> **Analyse Médicale · Machine Learning · R**
>
> Développement d'un modèle prédictif basé sur le dataset UCI Cleveland pour identifier les facteurs de risques cardiaques.
> * **Statistiques :** Tests de corrélation (Khi-deux, Shapiro-Wilk, Wilcoxon/Student).
> * **Modélisation :** Régression logistique optimisée (critère AIC).
> * **Performance :** Évaluation rigoureuse via Courbe ROC et Matrice de Confusion.

#### 🌿 [Économétrie : Analyse des ventes de Pesticides (2022)](https://github.com/Nika-Hetagourov/Analyse-des-ventes-de-produits-phytosanitaires-pesticides-en-2022-dans-trois-r-gions-fran-aises.)
> **Économétrie · Analyse Territoriale · R**
>
> Étude des disparités régionales (Normandie, Guadeloupe, Aura) dans l'usage des substances phytosanitaires.
> * **Modélisation :** Régressions linéaires (MCO) sur le log des quantités achetées.
> * **Tests Statistiques :** Validation via tests de Student, Fisher et Breusch-Pagan (hétéroscédasticité).
> * **Data Prep :** Fusion complexe de données régionales et répartition au prorata des substances.
